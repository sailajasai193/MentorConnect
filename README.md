# MentorConnect

**MentorConnect** is a **full-stack MERN (MongoDB, Express, React, Node.js) web application** that connects students with mentors. It allows students to schedule mentorship sessions, communicate with mentors, and get AI-powered guidance for better learning outcomes.

---

## Features

- Student and Mentor authentication
- Schedule, view, and manage mentorship sessions
- Email notifications for upcoming sessions
- Google Meet integration for virtual sessions
- AI-powered summarization for meeting notes
- Secure handling of API keys using environment variables

---

## Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)
- **APIs & Integrations:** Google Calendar, Google OAuth, AI APIs
- **File Handling:** Multer for file uploads
